# Proyecto de Búsqueda de Héroes de DC y Marvel en React

Este proyecto de React es una aplicación web interactiva que te permite explorar y buscar información sobre los héroes de DC y Marvel. Puedes descubrir detalles sobre tus personajes favoritos, filtrarlos  y encontrar toda la información que necesitas sobre ellos.

## Características principales

- **Búsqueda inteligente:**  La aplicación te permite buscar héroes específicos utilizando una barra de búsqueda intuitiva. Puedes ingresar el nombre del héroe y obtener resultados precisos al instante.

- **Exploración de héroes:** Puedes explorar una lista completa de héroes de DC y Marvel en una interfaz de usuario atractiva y fácil de usar. Desplázate a través de la lista para encontrar héroes conocidos y descubrir nuevos personajes.


- **Información detallada:** Al seleccionar un héroe, puedes acceder a información detallada sobre él. Descubre su biografía, habilidades, historias destacadas, equipos y más. Toda la información se presenta de manera clara y organizada para una fácil lectura.

## Tecnologías utilizadas

El proyecto se desarrolló utilizando las siguientes tecnologías:

React, Bootstrap, Animate.css
